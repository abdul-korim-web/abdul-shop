## 📌 Project Overview

ABDUL SHOP is a modern full-stack electronics eCommerce platform where users can browse and purchase

products like:
- 📱 Smartphones
-💻 Laptops
- 🎧 Electronic Gadgets
- ⌚ Smart Devices
- 🔌 Accessories

The platform includes:
---
- User authentication
- Product management
- Shopping cart system
- Admin dashboard
- Responsive UI
- Secure backend API

## ✨ Features

- 🛒 Add to cart system

- 📄 Dynamic product details page
- ⚡ Fast responsive UI
